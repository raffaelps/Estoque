Rest service para entidade de categoria do sistema SISTOQUE

Criar categoria

URI: <servidor_contexto>/rest/categoria
VERBO: POST
CONSUMO: Content-Type:application/json

Atualizar categoria

URI: <servidor_contexto>/rest/categoria
VERBO: PUT
CONSUMO: Content-Type:application/json

Apagar categoria

URI: <servidor_contexto>/rest/categoria
VERBO: DELETE
CONSUMO: Content-Type:application/json

Obter categoria

URI: <servidor_contexto>/rest/categoria/{codigo}
VERBO: GET
PRODUZ: Content-Type:application/json

Obter todas as categorias

URI: <servidor_contexto>/rest/categoria/list
VERBO: GET
PRODUZ: Content-Type:application/json

Estrutura do JSON de categoria

String: codigo
String: nome

Exemplo: {"codigo":"CL","nome":"Calçado"}
